# Single File ERP POS Demo

This example demonstrates how a basic ERP (Enterprise Resource Planning) and POS (Point of Sale) system can be implemented within a single HTML file. It showcases inventory management, a simple sales cart, and transaction logging using client-side JavaScript and `localStorage` for data persistence, embodying the article's concept of a self-contained, zero-backend application.

## Language

`html`

## How to Run

1. Save the code as `index.html`.
2. Open `index.html` in any modern web browser.
